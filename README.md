# Bürogummi Run – Office Dodger

Ein minimalistischer Endless-Runner mit drei Spuren: weiche Büro-Hindernissen aus und halte dich vom roten Radiergummi fern.

## Spielen

1. Repository herunterladen oder klonen.
2. `index.html` doppelklicken oder im Browser öffnen (kein Build-Schritt nötig).
3. Mit **A/D** oder **←/→** die Spur wechseln und möglichst lange überleben.

## Regeln
- 3 Leben, Kollisionen kosten jeweils ein Leben.
- Der Score steigt automatisch mit der Zeit.
- Nach dem letzten Leben erscheint das Game-Over-Overlay mit Score, Zeit und Call-to-Action.
