# Bürogummi Run – Office Dodger

Ein dreispuriger Endless-Runner im Büroflur: jetzt als schlanke 3D-Szene mit Three.js (lokal gebündelt), klaren Lanes und gut lesbaren Hindernissen.

## Spielen

1. Repository herunterladen oder klonen.
2. `index.html` im Browser öffnen (alles offline, kein Build-Schritt nötig).
3. Charakter wählen, mit **A/D** oder **←/→** die Spur wechseln und möglichst lange überleben.

## Regeln und Hinweise
- 3 Leben, Kollisionen kosten jeweils ein Leben.
- Der Score steigt automatisch mit der Zeit; Geschwindigkeit nimmt leicht zu.
- Startscreen und Game-Over-Overlay zeigen Score, Zeit und eine Call-to-Action zur KV-Lehre bei der Gemeinde Freienbach.
- Läuft komplett offline – Three.js steckt als lokales Bundle im Verzeichnis `js/`.
